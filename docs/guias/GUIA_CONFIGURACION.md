# Guía de Configuración Personal

Desde la pantalla de Configuración podés ver tu información, cambiar el tema visual del sistema y activar las notificaciones en tu celular.

---

## 1. Cómo acceder a la Configuración

En el menú lateral, buscá el ícono de **tuerca** o engranaje, o la opción que dice **"Configuración"**. Hacé clic ahí para entrar.

---

## 2. Ver tu información personal

En la parte superior de la pantalla vas a ver tus datos actuales:
- **Nombre**: Tu nombre completo.
- **Correo electrónico**: El correo con el que entrás al sistema.
- **Departamento**: El área a la que pertenecés.
- **Tipo de usuario**: Tu función en el sistema (Supervisor, Funcionario, etc.).

> Estos datos son de solo lectura en esta pantalla. Para cambiar tu nombre o contraseña, contactá al Administrador del sistema.

---

## 3. Cambiar el tema visual (claro / oscuro)

El sistema tiene dos modos visuales:
- **Tema claro**: Fondo blanco, ideal para ambientes con buena luz.
- **Tema oscuro**: Fondo oscuro, más cómodo en lugares con poca luz o de noche.

Para cambiar:
1. En la pantalla de Configuración, buscá la opción **"Tema"**.
2. Seleccioná **"Claro"** u **"Oscuro"** según tu preferencia.
3. El cambio se aplica de inmediato, sin necesidad de recargar.

---

## 4. Activar notificaciones en el celular

Para recibir alertas del sistema directamente en tu celular (aunque no tengas el sistema abierto):

1. Abrí el sistema desde el **navegador de tu celular** (Chrome, Safari, etc.).
2. La primera vez, el navegador te va a preguntar si querés recibir notificaciones. Tocá **"Permitir"**.
3. Si no apareció ese mensaje, entrá a **Configuración** en el sistema y buscá la opción **"Activar notificaciones push"** o **"Notificaciones del navegador"**.
4. Tocá el botón para activarlas y aceptá el permiso que te pide el celular.

A partir de ese momento, cada vez que te llegue una notificación del sistema (un ticket nuevo, un comentario, etc.) vas a recibir un aviso en la pantalla del celular.

### Si las notificaciones no llegan

- Verificá que el celular no tenga activado el modo **"No molestar"**.
- Asegurate de que las notificaciones para el navegador estén habilitadas en la configuración del celular (Ajustes → Aplicaciones → Chrome/Safari → Notificaciones).
- Si seguís teniendo problemas, probá desde otro navegador.

---

## 5. Opciones adicionales para administradores

Si sos administrador, en esta misma pantalla podés acceder a opciones avanzadas del sistema como:
- Configuración de correos electrónicos de notificación.
- Herramientas de mantenimiento del sistema.

Esas opciones solo son visibles para usuarios con perfil de administrador.
